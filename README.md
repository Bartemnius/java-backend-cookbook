# Java Backend Cookbook

Java Backend Cookbook is a practical, simple-English learning and interview preparation book for Java backend developers.

The goal is to organize core Java, Spring, databases, networking, architecture, DevOps, and interview concepts in one place.

## What This Repository Contains

- Markdown chapters for the book
- A working checklist for the full curriculum
- GitHub Pages configuration for a browsable website
- Future PDF/e-book source material

## Website

This repository is prepared for GitHub Pages using Jekyll and the Just the Docs theme.

Recommended GitHub Pages setup:

1. Push this repository to GitHub.
2. Open repository settings.
3. Go to `Pages`.
4. Set source to `Deploy from a branch`.
5. Select the `main` branch and the `/docs` folder.
6. Save.

GitHub will build the website from the Markdown files in `docs/`.

## Working Style

The book is written in simple English.

Each important topic should explain:

- what the concept is,
- why it exists,
- how it looks in code,
- what interviewers usually ask,
- what developers often misunderstand,
- what mid/senior developers should know.
